# Skillora Mobile App

Skillora is an Android mobile application developed using Kotlin and Jetpack Compose.

## Project Description

Skillora helps users find skilled professionals and also allows users to offer their own skills or services through the mobile app.

## Main Features

- User registration
- User login
- Firebase Authentication
- Firebase Realtime Database
- Skill/service browsing
- Notification screen
- Modern Jetpack Compose UI

## Technologies Used

- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Realtime Database
- Android Studio
- GitHub

## Database

This app uses Firebase Realtime Database to store user and app data.

## Authentication

Firebase Authentication is used for user login and registration.
